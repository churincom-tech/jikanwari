# Timetable Validation Checklist

## ハード制約
- [ ] 教員の同時刻重複がない
- [ ] 特別教室の同時刻重複がない
- [ ] クラスの同時刻重複がない
- [ ] 教科ごとの必要時数を満たす
- [ ] 勤務不可時間に授業がない
- [ ] 固定枠が変更されていない
- [ ] 固定授業が授業情報の週時数を超えていない
- [ ] 2時間連続必須の授業が隣接する2時限で配置されている
- [ ] 2時間連続必須の固定授業が片方だけ固定されていない
- [ ] 各クラスの途中時限に空きがなく、空きは日の末尾に寄っている

## ソフト制約
- [ ] 主要教科の曜日や時間帯の偏りが説明できる
- [ ] 同一教科の同日集中が過度でない
- [ ] 教員の空き時間や連続授業が極端でない
- [ ] 特別教室利用が極端に集中していない
- [ ] 非常勤教員の出勤日が不必要に増えていない
- [ ] 2時間連続が望ましい授業の未連続理由が説明できる

## 記録
検証結果は `artifacts/verification/` に残します。
