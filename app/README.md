# コマいぬ

完全無料のローカルWebアプリです。有料API、外部API、外部CDN、npm依存、ビルド工程は使いません。

## 起動方法
`app/index.html` をブラウザで開きます。

## 初期機能
- 「時間割サンプル例」からサンプル時間割候補をすぐ確認
- 左側の手順ナビに沿った入力
- 学年、クラス数、曜日、時限数の入力
- 教科ごとの週時数、担当教員、特別教室の入力
- 表をクリックして入れる固定授業と勤務不可時間の入力
- 2時間連続必須、2時間連続が望ましい授業の設定
- 教育課程マスタによる標準時数、教科名、地域独自教科の調整
- 固定授業が授業情報の週時数を超えない入力制限
- 候補生成前の条件確認
- 最大3件の時間割候補生成
- クラス別時間割表示
- ハード制約違反数、ソフト制約スコア、改善候補の表示
- JSON保存とJSON読込

## 注意
この試作はブラウザ内だけで処理します。実データを使う場合も、ユーザー操作でJSON保存しない限り外部へ送信しません。
