# Timetable Domain Notes

## 基本概念
- 学校: 曜日、時限、学年、クラス、校時、行事を持つ単位
- 学年: 1年、2年、3年など
- クラス: 1年1組など、時間割を割り当てる基本単位
- 教員: 担当教科、担当クラス、勤務不可時間、非常勤条件を持つ
- 教科: 週時数、連続授業可否、1日複数回可否、教室条件を持つ
- 教室: 普通教室、理科室、音楽室、体育館、PC室など
- 時限: 曜日と校時の組み合わせ
- 固定枠: 学活、道徳、総合、会議、行事など変更しない枠

## 時間割生成で重要な観点
- まずハード制約を満たす
- その後、ソフト制約をスコアで改善する
- 手修正できる余地を残す
- 生成結果を検証可能にする
- 教務担当者が理由を理解できる表示にする
