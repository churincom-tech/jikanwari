# Context Collection Guide

## 目的
ユーザーに大量質問せず、時間割作成に必要な情報を優先順位つきで集めます。不明点は確定せず `context/assumptions.md` に記録します。

## 最初に集める3点
1. 学年、クラス数、曜日、時限数
2. 教科ごとの週時数、担当教員、特別教室の有無
3. 固定授業、勤務不可時間、必ず守りたい条件

## 任意で追加すると品質が上がる資料
- 現在使っている時間割表
- 教員別の担当一覧
- 教室利用ルール
- 昨年度の時間割で困った点
- 表計算ファイルやCSVの既存データ

## 後回しでよい情報
- 認証方式
- 複数校管理
- 高度な権限設計
- PDF出力の細部
- 印刷レイアウトの最終調整

## 収集時のルール
- 専門用語を押しつけない
- 不明点を一度に大量に聞かない
- 仮定で進める場合は仮定として明記する
- 個人情報を含む資料は外部送信しない
- まずサンプルデータで検証可能な粒度を目指す
