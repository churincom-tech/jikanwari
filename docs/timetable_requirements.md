# Timetable Requirements

## 機能要件
- 学校設定を入力できる
- 教員、教科、クラス、教室を管理できる
- 固定授業と勤務不可時間を指定できる
- ハード制約とソフト制約を分けて設定できる
- 学年、クラス数、曜日、時限数を画面から入力できる
- 教科ごとの週時数、担当教員、特別教室の有無を画面から入力できる
- 固定授業、勤務不可時間、必ず守りたい条件を画面から入力できる
- 時間割案を自動生成できる
- 複数の時間割候補を生成できる
- 候補ごとに制約違反、スコア、改善候補を表示できる
- 生成結果の制約違反を検出できる
- 生成結果を手修正できる
- CSVまたはJSONで入出力できる

## 非機能要件
- ICTに詳しくない利用者でも操作できる
- 生成と検証の理由が追える
- サンプルデータで再現可能に検証できる
- 初期試作は軽量に動作する
- 後から制約や評価指標を追加できる
- 入力途中でも不足項目や矛盾が分かる
- 候補表示では、最良候補だけでなく比較理由も分かる
