# Prototype Scope

## 最小試作で扱うこと
- 完全無料でローカルブラウザ上だけで動く静的Webアプリ
- 画面から学年、クラス数、曜日、時限数を入力する
- 画面から教科ごとの週時数、担当教員、特別教室の有無を入力する
- 画面から固定授業、勤務不可時間、必ず守りたい条件を入力する
- 入力内容をもとに時間割候補を生成する
- 複数候補を一覧またはタブで表示する
- ハード制約チェック
- 基本的なソフト制約スコア
- クラス別時間割の表示
- 入力内容と生成結果のJSON保存またはエクスポート

## 後続に回すこと
- 認証
- 複数校管理
- 高度な権限管理
- 完成度の高いPDF出力
- 複雑な年間計画
- 外部システム連携

- 高度な手修正履歴

## 試作完了の条件
ユーザーが画面から必要情報を入力し、ハード制約違反なしの時間割候補を少なくとも1案表示でき、候補ごとの検証結果を画面またはログで確認できることです。
